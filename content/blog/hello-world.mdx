## 简介

MDX 是一个让你在Markdown文件中写入React组件的库。这意味着你可以使用Markdown的所有功能，同时还能添加交互式组件，比如按钮、弹窗等。

## 按钮组件示例

下面是如何使用`theme-ui`中的`Button`组件：

```jsx
<Button variant="primary">点击我</Button>
```

## 代码块

MDX也支持标准的Markdown代码块，例如：

```jsx showLineNumbers
function Greeting() {
  return <p>Hello, MDX!</p>;
}
```
